# 今天也要守住

《今天也要守住》是一款本地单机 Demo 游戏，使用 Cocos Creator 3.8.8 + TypeScript 制作。

## 当前 Demo 内容

- 战斗主场景
- HomeBase 精神值
- 怪物生成、移动、受击与死亡
- 玩家自动攻击与子弹
- 1-5 波基础流程
- 技能选择界面
- Victory / GameOver 战报卡
- 音效与背景音乐

## 本地打开

1. 安装 Cocos Creator 3.8.8。
2. 用 Cocos Creator 打开：
   `cocos-project/today-guard-game`
3. 打开场景：
   `assets/scenes/BattleScene.scene`
4. 运行预览。
